Bvlib库开源，正在建设。希望每天写一点，积少成多。

一、运行环境
目前仅支持linux环境

二、Bvlib结构

    Bvlib
    |------include  存放头文件
    |------lib      存放生成的库文件libBvlib.so
    |------bin      生成的可执行文件(测试的可执行程序)
    |------src      源码目录
    |------example  存放用例程序
            |------threadPool  线程池示例
    |------Makefile 编译Makefile
    |------readme.txt   简单介绍

三、支持的功能
1，链表
2，队列
3，线程池


四、规划功能
1，日志
2，时间
3，socket通信
